Variables = Son cajitas las cuales poseen dentro de ellas informacion la cual nos permite guardar datos y escribir sobre ellos
String = es un tipo de dato el cual se muestra como una cadena de caracteres, palabras, etc. Este se muestra entre comillas
Declaraciones = son condiciones que te separan una cosa de otra, un ejemplo puede ser 'si yo tengo mas de 90 a√±os, soy viejo, pero si yo tengo menos de 90, soy joven' (obviamente no es verdad esto) de esta forma da por cierto una condicion y denega otra, siempre estaras parado en un unico valor para determinar esto
Valores booleanos = son 2 tipos de datos, que permiten 2 tipos de respuestas posibles, 'verdadero' o 'falso' un ejemplo seria 'si yo tengo la luz prendida seria verdadero, pero en caso de estar apagada, seria falso' son respuestas muy diferentes una de otra
Funciones = Las funciones estan hechas para no repetir muchas veces las mismas variables, es como un atajo para cruzar la calle, algo mas rapido, mas efectivo. La funcion tiene una cajita a dentro que tiene el nombre de 'nuevoscope', dentro de esta cajita tiene codigos pero uno muy importante es el 'return' con este podes hacer que la cajita 'nuevoscope' responda a lo que pide la funcion
